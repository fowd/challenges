A9.1 - Navigation Style

Goal

To use CSS to transform the look of your Navigation content.

Description

For this assignment, you'll continue to use the interior.css stylesheet to transform your unordered list into a horizontal navigation element.

HTML

Document: html.html

In your unordered list's list-item, add the following class attribute — only to the HTML li opening tag
<li class="current"><a href="html.html">HTML</a></li>

Document: css.html

In your unordered list's list-item, add the following class attribute — only to the CSS li opening tag
<li class="current"><a href="css.html">CSS</a></li>
