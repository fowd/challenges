A13.1 - jQuery Gallery

Purpose

The purpose of this assignment is to introduce you to the world of jQuery—a framework for javascript. You will learn how to develop a simple gallery lightbox using javascript and the jQuery library.

Description

Wether you are a photographer, graphic designer, or even a business wanting to showcase your work, the "lightbox" method has become a very popular way of showcasing images. Lightbox is a method used to display large images using modal dialogs. On a Lightbox-enabled page, a user can click an image to have it magnified in a Lightbox window, which resizes itself according to the size of the image using a gliding animation. jQuery helps this process to become even easier do and there's plenty of options to choose from.

While there are dozens of lightbox scripts to choose from, two of the easiest scripts are

  • fancyBox by Jānis Skarnelis
  • jQuery Lightbox Plugin by Leandro Vieira Pinho

Both, at the basic core, work very similarly, but, in our case, we'll use the FancyBox. There is a video tutorial provided for the jQuery Lightbox Plugin on module home page.

Requirements

Download Assignment File

  • Download the file: a13-lastname
  • Uncompress
  • Rename the folder to have your lastname (e.g. a13-michaud)

Images:

I've provided your three images to start with, but you will need to add three more of your own (your own wok or images from some other source).

  • Your thumbnail images should be consistent with the thumbnails already provided (150px wide x 100px tall)
  • Your large images should be no taller than 680 pixels and no wider than 1024 pixels

HTML:

  • Create a basic HTML document (just like we've been doing). Make sure you have:
    • The META element that has name="author" and content="your name".
    • Placed text in the TITLE element (e.g. The Gallery of Thomas Michaud).
  • Add the SCRIPT element for jQuery and fancybox
  • Add the LINK element to the css document for fancybox
  • Add another lINK element for your own css document (name it: gallery.css)
  • CONTENT: Place the images in an unordered list (UL and LI tags required)
    • Make sure the opening UL (<ul> tag has the class attribute with the value of "gallery"
    • You can add a title attribute to the large image, if you want text below the image.
  • Make sure you place the last SCRIPT element (with the document ready function) just above the closing </body> tag
  • Make sure you validate your HTML!!

CSS:

Using CSS, Make sure the image thumbnails are shown in a horizontal row.
