Final Project Submission

Description & Requirements

Here's the finish line! You are to now submit all your hard work as a finished website. Here's a checklist of required items:

  • For a more complete checklist, make sure you view the Final Project Checklist.
  • Make sure the code (HTML and CSS) validates
  • Are your images in the gallery a good size for web?
  • Test that your site shows (nearly) consistently in all modern browsers (Firefox, Safari, Chrome and (if you can) Try it out on Internet Explorer (7 or higher))
  • Make sure you contact form works (remember, you need to use the server for this) ... and that the user gets a "thank you" message; can I navigate to other parts of your site too?

Tools

HTML editor, Image editor, and any other tools to help finish the project.
