Working with CSS

In the site.css document you need to add the following rules...

Styling the background and text
Add a rule that has a body selector with the following declarations:
  • The background-color should be a gray with the value of #cccccc
  • The font-family should be set to Helvetica, Arial, sans-serif
All text heading levels (h1-h6) should have a font-family of Georgia, serif
  • Note: you need to use the combination selector (,) for the text headers
All p and li selectors should have a line-height of 1.25em
the p selector that has a child em selector should have a font-weight of bold
Set the ul selector with the ID selector (#) of nav to a background-color value of #dddddd
Styling Anchor Elements
Set the a selector to: #008b14
Set another a selector with the pseudo :link to #008b14
Set another a selector with the pseudo :visited to #649a00
Set another a selector with the pseudo :hover to #00638b
Set another a selector with the pseudo :active to #008b14
Set another a selector with the pseudo :focus to #00638b
Remove Old CSS
In the css.html document, delete the style element and the code changing the h2 to the color orange: 
<style type="text/css" media="screen">
h2 {color: orange;}
</style>
In the html.html document move (don't simply copy but cut) only the CSS into the site.css (but not style element)—remember to delete the style element. 
	
<style type="text/css" media="screen">
table, tr, th, td{border:1px solid black;}
th, td{ padding: 4px;}
</style>

Linking the Stylesheet

Last thing is to check to see if we can link our external style sheet to our HTML documents.

Using the link element
After the title element in each html document (css.html, html.html, and index.html), place a link element that finds your site.css document in the css folder.
