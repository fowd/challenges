M4.1 Midterm: CTA » Finalized

Description & Requirements

Code (HTML and CSS) your book or product promo. Make sure you have the following items:

  • Valid HTML document and well structured (using comments)
  • Valid CSS document and well organized (using comments)
  • Good readability (line-height and font-size)
  • Good design principles
  • Images in an images folder
  • A link to purchase the book or product (Amazon, Barnes and Noble, Borders, or some other store)
  • Synopsis, Reviews, Video, Author/Company Information

Purpose

To develop a website-promo for the book or product you've selected.

Tools

HTML editor and Image editor
